浏览器插件

发表于2022-02-24|更新于2022-04-14|tools
|字数总计:821|阅读时长:2分钟|阅读量:

简介

以下介绍的为edge或者chrome浏览器的官方插件，和油猴并非相同。edge和chrome操作相同，下面以edge为例。

下文将会介绍油猴及其他插件。

油猴：一款免费实用且强悍的浏览器插件，一般代指 tampermonkey。
适用于Chrome,Microsoft Edge,Safari,Opera Next, 和Firefox。

以下使用体验仅为个人体验，实际情况可以根据自身选用甚至设计。

mark

听说Zotero管理文献很不错，有机会会用的。

油猴使用教程

1. 打开 tempermonkey.net ,选择“添加至chrome”

Tempermonkey

https://www.tampermonkey.net/

2. 打开 Greasy Fork ，搜索脚本并添加即可

此网站支持中文，对国人十分友好。

Greasy Fork

https://greasyfork.org/zh-CN

3. 油猴脚本为 js 写成，如有所需，浅学 js 即可更改脚本，并且油猴支持自定义脚本

如笔者曾将“隐藏百度广告”的脚本进行修改，部分脚本大佬可以写出抢课脚本等等。

油猴插件推荐

AC-百度去广告

  1. 这个会先把右边“热点”和“相关人物”一栏给去掉，使得看起来更加整洁美观。
  2. 其次对于主页面的广告，通过判定ID等方式来确定广告并移除之。
  3. 笔者修改了之后只保留了如下代码，略有js基础的读者都能看懂。

百度文库下载&复制

网页解除限制

文本选中复制

可以支持自定义目标网页，但根据实测，部分网站仍不能解除复制限制。

智慧树知到网课助手

这款应该是大学生喜闻乐见的。

Userscript+ : 显示当前网站所有可用的UserJS脚本

这款是给当前页提供可用的脚本的一款脚本。

其他使用教程

1. 打开 edge 的 扩展 板块

2.点击 获取 Microsoft Edge 扩展

Tempermonkey

edge://extensions/

其他途径：*.crx格式文件是Google Chrome谷歌浏览器扩展应用插件，只需搜到相应的crx文件拖入扩展程序窗口即可。

插件推荐

oneTab

Tempermonkey

https://microsoftedge.microsoft.com/addons/detail/onetab/hoimpamkkoehapgenciaoajfkfkpgfop?hl=zh-CN

绝对强推

号称可以节省浏览器 95% 的内存占用。 edge 和 chrome 常年处于内存占用榜的前几位，岂可修。
虽然笔者实测并不可以，但是好歹也能够节省个好几百M。

NeatDownloadManager

Tempermonkey

https://microsoftedge.microsoft.com/addons/detail/neatdownloadmanager-exten/pbghcbaeehloijjcebiflemhcebmlnke?hl=zh-CN

如果你知道什么是idm，但是苦恼于没有途径获取idm，那ndm就是你最好的选择！

Scientific surfing

插件名称：setupVPN
敏感话题，此处跳过。

Listen 1

放歌整合插件，说是插件，其实和网站也差不多，使用频率也不高。

Pxxxxxx 下载

笔者没用过此插件，仅仅耳闻，如有看官了解可以私戳笔者。

JsonView

前端后端开发的时候都会用到的吧。当然还有其他替代品，选一个就行。
